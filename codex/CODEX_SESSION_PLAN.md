# Codex Session Plan

## Session 1 — Scaffold and design tokens

Build the Next.js TypeScript scaffold, global layout, CSS variables, and base design shell. Commit once the app runs with placeholder content.

## Session 2 — Content loader

Implement Markdown discovery, frontmatter schemas, validation, registry generation, navigation tree generation, and plain-text extraction. Add tests or validation scripts for duplicate routes and missing metadata.

## Session 3 — Markdown renderer

Implement the Markdown renderer with GFM tables, code blocks, heading anchors, Mermaid blocks, details/summary support, and safe raw HTML handling.

## Session 4 — Navigation UI

Implement the topbar, brand lockup, domain tabs, domain header, secondary subnav, responsive layout, and section pages. Verify all migrated content routes render.

## Session 5 — Search and full archive

Generate search index, implement search UI, no-results state, and full archive view. Add print styles.

## Session 6 — Hardening and template docs

Run lint/typecheck/build, fix performance issues with large archive content, write template README, document content authoring workflow, and verify a project can be created by replacing `/content`.
