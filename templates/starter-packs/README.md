# Starter Packs

These packs are copyable starting points, not separate apps. Each pack includes:

- a `site.config.example.ts`
- a small `content/` root with example routes
- a short README explaining what the pack is for
- content types and structure that fit the starter

## Available Packs

- [AI Product Documentation](./ai-product-documentation)

## How To Choose

1. Pick the pack that matches the kind of source-of-truth you are building.
2. Copy the starter folder into a fresh repo.
3. Rename `site.config.example.ts` to `site.config.ts` and edit the values.
4. Copy the `content/` tree into your project content root.
5. Replace the sample content with your own pages while keeping the same route structure.
